QuickLaunch
RP Soft 2002
WIN / DIVERS
GPL
Cet ensemble de programme a pour but d’optimiser l’utilisation des touches de lancement rapide que l’on peut trouver sur les portables (par exemple les Gericom).Ce programme permet en particulier d’envoyer n’importe quelle commande à une application précise, par exemple pour dire à Winamp de jouer le morceau suivant. La configuration se fait à l’aide d’un fichier de configuration simple dont voici un exemple :

[Button1]
command=C:Program FilesInternet ExplorerIEXPLORE.EXE
params=
show=

[QuickLaunch]
command=
findWindow=Winamp v1.x
wmCommand=40048
params=
show=

Les programmes sont capables de :

  • Lancer n’importe quelle application avec n’importe quel argument
  • Transmettre n’importe quelle commande à une application identifiée par son nom de fenêtre (ex : Winamp)

Manipulation des liens Windows
RP Soft 2002
WIN / DIVERS
GPL
Cet ensemble de programme est destiné à faciliter la manipulation des liens symboliques Windows (*.lnk) depuis un script, permettant ainsi l’automatisation de modifications sur ces liens.Plusieurs fonctions sont disponibles :

  • linkget : récupère les informations sur un lien.
  • linkset : créé un lien.
  • linkresolve : tente de mettre à jour automatiquement la cible.
  • linkupdt : modifie l’emplacement pointé.

RenDate – Renommer un fichier en fonction de la date
RP Soft 2002
WIN / DIVERS
GPL
Ce programme a pour but de renommer facilement un fichier avec une information de date depuis un fichier batch. (exemple typique : renomme monfichier.txt en monfichier_030317.txt)

Usage : RenDate [/m] [/n] [/v] [/f ] 

    filename  : the name of the file to be copied/moved.
    /m        : move the file instead of creating a copy.
    /n        : do not use the filename provided as base filename.
    /v        : verbose mode.
    /f format : User Defined Format
                 the most common flags are : (see strftime for the full list)
                    %c Date and time representation appropriate for locale
                    %d Day of month as decimal number (01 - 31)
                    %H Hour in 24-hour format (00 - 23)
                    %j Day of year as decimal number (001 - 366)
                    %m Month as decimal number (01- 12)
                    %M Minute as decimal number (00 - 59)
                    %S Second as decimal number (00 - 59)
                    %y Year without century, as decimal number (00 - 99)
                    %Y Year with century, as decimal number
                    %% Percent sign
                 the default is : _%Y%m%d (appends _YYYYMMDD)

Cluster
RP Soft 1999
Win9x / SYSTEME
Freeware
Ce programme peut servir dans le cas où vous souhaiteriez simuler ce que prendrait le répertoire dans un autre système de fichiers qui a des tailles de clusters différentes.Par exemple, vous souhaitez créer un ZIP et vous créez l’arborescence correspondante sur votre disque dur et essayez de ne pas dépasser la taille maximum, en s’en approchant le plus possible…

Mais attention 2 problèmes se posent :

  • Tout d’abord quand vous regardez la taille des fichiers il faut tenir compte de la taille en octets “utilisés” selon l’appellation de l’explorateur, cela désigne en effet l’espace occupé par les clusters occupés.
  • Ensuite, le support pour lequel vous destinez ces fichiers n’a pas forcément la même taille de cluster que votre disque dur (c’est même peu probable). Dans ce cas l’information est fausse. Deux possibilités se présentent : Le cluster est plus grand,
    donc les fichiers vont prendre plus de place et ce que vous avez préparé consciencieusement ne pourra pas tenir sur le support de destination ! C’est le pire cas. Si le cluster est plus petit, il vous restera de la place contrairement à ce que vous pensiez…

Voila les difficultés auxquelles Cluster prétend répondre.

WinFont
RP Soft 1998
WIN / POLLICES TTF
OpenSource
Cet utilitaire est destiné aux personnes ayant un grand nombre de polices de type TrueType TTF, et voulant mettre de l’ordre sans en faire les frais.  Le problème avec tous les gestionnaires de Polices que j’ai vu, c’est qu’ils ne savent pas travailler en masse
avec des polices non installées (visualisation, création de listes, de tables de caractères, et informations…) Je ne sais pas non plus le faire.Ce programme donne juste la liste des polices d’un répertoire donné, en donnant pour chaque :

  • Le nom du fichier TTF
  • Le nom de la police
  • La Taille
  • Le Type
  • …

Il est basé sur l’ancien système de police Win31, mais marche sur Win95 grâce  à la compatibilité des fonctions.  Il a pour principe de créer temporairement pour chaque TTF un
.FOT   (sorte d’installation, mais tout à fait transparent et non polluante)  et d’extraire de ce FOT les informations que j’ai pu identifier a vue de nez.

Utilisation :

Ce programme est dénué de toute interface digne de son nom.

Il vous demande :

  • le chemin du répertoire a examiner
  • le chemin du répertoire de travail (pour les FOT temporaires)
  • le nom du fichier a créer, qui contiendra les infos.

Le format du fichier est fait pour pouvoir etre récupéré par un tableur, avec les séparateur “;”
