Init
RP Soft 1995
DOS / SYSTEME
OpenSource
Init enregistre dans un fichier externe tous les jours et dates lorsqu’il est lancé. Cela vous permet, si vous le mettez dans votre autoexec.bat de savoir les horaires d’allumage de votre PC, et ainsi de contrôler a posteriori l’accès à votre PC.

NoHTML
RP Soft 1999
DOS / HTML
OpenSource
NoHTML a pour but d’effacer sans pitié tous les tags HTML, pour en extraire la substantifique moelle… Le TEXTE brut (et lisible)  !!!!Il a été conçu pour ceux qui n’ont pas ou ne veulent pas s’encombrer d’un  visualisateur HTML mais qui veulent avoir une idée de ce qui se cache dans cette page.

Or parfois elle est trop encombrée par ces tags (si elle est générée  automatiquement).

Utilisation :
NoHTML <NomFic>

Il efface tout ce qu’il trouve entre 2 <…>

Problèmes: réactions délicates dans le cas de GnaGnaGna
<TagTag<Commentaire>BugFinTag>

Dans ce cas il s’arrête au premier (c’est logique et c’est tout)
Ce programme est très rudimentaire, mais peut rendre service et c’est son but.

SaveMEM
RP Soft 1998
DOS / SYSTEME
OpenSource
SaveMem est utilitaire rudimentaire, mais qui peut être d’un grand secours. C’est un programme résident, qui lorsque vous l’activerez, sauvegardera tout le contenu de votre mémoire (les 1 Mo accessibles directement) dans un fichierUtilisation :

  • Lancer SaveMem.Com pour l’installer en mémoire.
  • Appuyer sur la touche IMPR-ECRAN pour faire une sauvegarde.

Conseils et Remarques :

  • Ce programme est très rudimentaire, donc à manipuler avec attention.
  • Il n’y a pas de module de désinstallation prévu (c’est plus fiable).
  • Il n’y a pas d’appel à l’ancien gestionnaire de l’interruption 05,  donc la touche IMPR-ECRAN ne servira plus que à SaveMem (question de  stablité)
  • Il est évident qu’il faut mieux prévoir l’espace disque suffisant avant  activation… (sinon je n’ai pas testé ses réactions)
  • Le fichier est SAVEMEM.BIN une fois pour toutes. Si vous voulez le changer  éditer le fichier .ASM ou même le fichier .COM (le nom doit être ASCIIZ)
  • Ce programme est Résident. A utiliser exclusivement sous DOS.

Ce programme est très rudimentaire, les plantages ne sont pas exclus, bien que je n’en ai jamais observé. Mais vous devez l’utiliser en connaissance de cause : Sauvegardez tout ce que vous avez d’ouvert. (A ce propos, faites attention a SmartDrive et son cache en écriture…)

SysPause
RP Soft 1995
DOS / SYSTEME
OpenSource
Ce programme permet de marquer une pause lorsqu’il est appelé. Sa particularité est qu’il peut faire une pause dans les instructions du Config.Sys pour détecter une éventuelle erreur ou contrôler les affichages souvent  illisibles (rapidité et défilement).Utilisation :

  • Dans le Config.Sys : Mettre cette ligne aux endroits voulus 
          DEVICE = SYSPAUSE.EXE
  • Il peut aussi servir (a titre de gadget) en mode normal, en tapant  à l’invite DOS ou dans un fichier BAT 
          SYSPAUSE.EXE

Conseils et Remarques :

  • Je l’ai écrit en voulant savoir comment faire un programme dans le  config.sys qui sache être appelé en ligne de commande (comme EMM386.EXE …)
  • A priori, cela devrait marcher sous Win95 (pas testé)
  • Le message peut être modifier dans le fichier .ASM ou directement dans  le fichier .EXE
